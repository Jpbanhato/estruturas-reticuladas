# Análise e Cálculo de Modelos Estruturais Reticulados Bidimensionais

Este repositório contém o código utilizado para a **análise e cálculo de modelos estruturais reticulados bidimensionais**, compostos por **barras conectadas por nós**, considerando elementos com **seção constante**.

## Características do Projeto

- Implementação de **modelos estruturais reticulados 2D**  
- Cálculo e análise de **estruturas formadas por barras e nós**  
- **Seções constantes** aplicadas aos elementos estruturais  

## Descrição do Funcionamento

Além do código-fonte, este repositório inclui uma **descrição detalhada do funcionamento do programa**, abordando:  

- **Validação dos exemplos utilizados**  
- **Pontuais limitações do modelo**
